# Automation-test

This repository contains multiple automation projects implemented using Selenium WebDriver, Playwright, Reqnroll, NUnit, and RestSharp.

The projects demonstrate UI automation, API automation, BDD automation, data-driven testing, JSON schema validation, API integration with UI tests, and scalable automation framework design.

### ## Design Patterns & Architecture

* Page Object Model (POM): Encapsulates UI interactions

* Service Layer: Encapsulates API operations

* Workflow Layer: Encapsulates business flows

* Fixture Pattern: Manages shared test context

* Data-Driven Pattern: Separates test logic from test data

* Schema Validation Pattern: Verifies API contract integrity
